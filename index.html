如何使用？

在 X Layer 上管理你的 .OKX 子域名 (.xxx)！

  • 连接钱包：使用 MetaMask 连接到 X Layer（链 ID: 196）。确保有足够的 OKB 用于 gas 费用和 X0 用于铸造。
  • 创建子域名合约：输入你的 .OKX NFT Token ID 和铸造价格（X0），创建子域名合约。
  • 铸造子域名：输入 .OKX NFT Token ID 和子域名 ID（例如 test）。如需支付，需先授权 X0 代币，然后铸造。
  • 检查铸造状态：验证子域名 ID 是否已被铸造（大小写不敏感）。
  • 安全与唯一性：仅 .OKX NFT 持有者可创建合约，每个子域名在合约内唯一。
